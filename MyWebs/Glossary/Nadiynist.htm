6. Характеристики надійності і технічного обслуговування

Максимальна тривалість обслуговування з використанням засобів технічного обслуговування при одночасній роботі трьох осіб і температурі навколишнього повітря не нижче 288 К (15 ° С), ч:

 

- для контрольного огляду (КО)..........................

0,2

- для щоденного технічного обслуговування (ТО)

1,6

- для технічного обслуговування після перших 2000 км пробігу.......................................

4,5

- для технічного обслуговування № 1 (ТО № 1)

(через кожні 2000 км пробігу)......................................

2,0

- для технічного обслуговування № 2 ( ТО № 2)

(через кожні 6000 км пробігу)......................................

6,5

- для технічного обслуговування, що виконується через кожні два ТО № 2 (через кожні 12000 км пробігу)..............................................

9,5

- для регламентно-технічного обслуговування, що виконується після 10 років зберігання....................................................

 

15,0

Допустимий термін зберігання без змін технічних характеристик з проведенням регламентних робіт, років

 

10, 0
